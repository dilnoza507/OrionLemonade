# Техническое задание (ТЗ) — Система учёта и управления лимонадным цехом

> Извлечено из файла: ТЗ Лимонад.docx

---

## 1. Общая информация

**Цель проекта:** разработка системы, которая позволит контролировать производство лимонада, склад, расходы, продажи и расчёт заработной платы сотрудников.

**Пользователи системы:**
- Владелец / директор
- Бухгалтер
- Менеджер по продажам

---

## 2. Модуль «Рецептура и производство»

### 2.1 Справочник рецептур
Для каждого вида лимонада хранить:
- Название продукта
- Объём партии (л / бутылки)
- Состав:
  - Ингредиент
  - Количество
  - Единица измерения
- Себестоимость партии (автоматически)
- Себестоимость за 1 литр / бутылку

### 2.2 Производственный учёт
- Создание производственной партии
- Выбор рецептуры
- Автоматическое списание ингредиентов со склада
- Фиксация даты производства
- Фиксация количества готовой продукции
- Учёт брака

---

## 3. Модуль «Склад»

### 3.1 Склад сырья
Хранить по каждому ингредиенту:
- Название
- Категория (сахар, вода, ароматизатор и т.д.)
- Остаток
- Единица измерения
- Закупочная цена
- Минимальный остаток

**Функции:**
- Приход сырья
- Списание (производство, порча)
- Контроль остатков
- Уведомление о минимальном остатке

### 3.2 Склад готовой продукции
- Название продукта
- Остаток
- Дата производства
- Срок годности

---

## 4. Модуль «Расходы»

### 4.1 Виды расходов
- Сырьё
- Зарплата
- Аренда
- Коммунальные услуги
- Транспорт
- Прочие расходы

### 4.2 Учёт расходов
Для каждой операции:
- Дата
- Статья расхода
- Сумма
- Комментарий
- Привязка к периоду

**Отчёты:**
- Расходы за день / месяц
- Расходы по статьям

---

## 5. Модуль «Продажи»

### 5.1 Учёт продаж
- Дата продажи
- Клиент
- Продукт
- Количество
- Цена за единицу
- Общая сумма
- Способ оплаты (нал / безнал)

### 5.2 Отчёты по продажам
- Продажи за период
- Продажи по продуктам
- Валовая прибыль
- Чистая прибыль (продажи – расходы)

---

## 6. Модуль «Зарплата работников»

### 6.1 Справочник сотрудников
- ФИО
- Должность
- Зарплата

### 6.2 Учёт рабочего времени
- Дата
- Сотрудник
- Количество отработанных часов / смен

### 6.3 Расчёт зарплаты
- Автоматический расчёт за период
- Учет премий и штрафов
- Итоговая сумма к выплате

### 6.4 Отчёты
- Зарплата по сотрудникам
- Фонд оплаты труда за месяц

---

## 7. Итоговые отчёты
- Себестоимость продукции
- Прибыльность каждого вида лимонада
- Общий финансовый результат
- Остатки склада
